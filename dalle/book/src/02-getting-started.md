# Getting Started

## Requirements

- Go 1.20 or newer
- macOS, Linux, or Windows

## Installation

Add **trueblocks-dalle** to your project with:

```sh
go get github.com/TrueBlocks/trueblocks-dalle@latest
```

No additional setup is required. All dependencies are managed via Go modules.

## Importing

